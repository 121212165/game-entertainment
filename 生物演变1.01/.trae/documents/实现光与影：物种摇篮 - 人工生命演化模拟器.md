# 实现计划

## 1. 项目初始化
- 使用 Next.js 15 + TypeScript 5 创建项目
- 安装必要依赖：Tailwind CSS、shadcn/ui、Recharts、Canvas API

## 2. 核心架构设计

### 2.1 类型定义与接口
- 创建 `types.ts` 定义核心类型（坐标、地形、生物属性等）
- 设计 `Organism` 接口作为所有生物的基类

### 2.2 环境系统
- 实现 `Environment` 类管理 50x50 网格世界
- 设计地形生成算法（平原、森林、光泉）
- 实现资源再生机制（光尘生成与扩散）

### 2.3 生物系统
- **Glimmer 类**：实现光能体的行为逻辑
  - 能量管理、智能防御、闪烁逃生
  - 森林之子遗传印记机制
- **Gloomer 类**：实现暗影体的行为逻辑
  - 饥饿驱动的狩猎行为
  - 狂暴模式机制

### 2.4 演化引擎
- 实现生物繁殖与遗传算法
- 设计环境印记遗传规则
- 实现基因突变机制

## 3. 渲染与可视化

### 3.1 Canvas 渲染系统
- 使用 `useRef` 和 `useEffect` 管理 Canvas
- 实现高效的网格渲染
- 设计生物可视化（不同形状和颜色区分）

### 3.2 数据可视化
- 使用 Recharts 创建种群动态图表
- 实现属性演化趋势图
- 设计生态系统状态面板

## 4. 用户界面

### 4.1 主界面布局
- Canvas 世界视图居中显示
- 控制面板（暂停/继续、单步执行、重置世界、速度调节）
- 可折叠的数据图表面板

### 4.2 交互功能
- 鼠标悬停显示生物详细信息
- 实时调整模拟参数
- 保存/加载模拟状态（可选）

## 5. 优化与测试
- 性能优化（减少 Canvas 重绘、优化空间查询）
- 代码测试与调试
- 确保模拟稳定性和可玩性
